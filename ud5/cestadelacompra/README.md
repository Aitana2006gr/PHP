# Proyecto "CestaDeLaCompra"
Este proyecto es para la utilización de sesiones para almacenar la información del usuario

## Base de datos: dwes
Uso la clase "Conexion.php" para conectarme

## Páginas:
- Login (login.php)
- Listado de productos (productos.php)
- Cesta de compra (cesta.php)
- Pagar (pagar.php)
- Logoff (logoff.php)
